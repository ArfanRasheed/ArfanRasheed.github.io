Software and systems projects from my academic and personal development work. All source code is available in my GitHub repository.

Honors College Thesis: Zero-Knowledge Proofs

For my Honors College thesis, I delved into the fascinating world of zero-knowledge proofs, a cryptographic method that allows one party to prove knowledge of a fact without revealing the fact itself. Using Python and the NetworkX library, I designed and simulated real-world authentication scenarios to explore secure data transmission methods. This project not only deepened my understanding of cryptographic protocols but also taught me how to approach complex problems analytically. The experience of writing and presenting my thesis was invaluable in refining my technical communication skills. You can view my thesis here.

2D Platform Fighter Video Game

As part of a team, I worked on developing a 2D platform fighter game over six months using agile methodologies. My role involved designing game mechanics and collaborating on feature integration. I learned how to handle challenges like collision detection and character movement while ensuring smooth gameplay. This project was a great opportunity to improve my teamwork and problem-solving skills, as well as to experience the iterative process of game development. Seeing the final product come together was incredibly rewarding.

Speedrunning Game with Java and Multithreading

For this project, I developed a speedrunning game that tracked player times using a multithreaded Java system. The game featured real-time updates for player performance, including current times, fastest records, and new achievements. Implementing multithreading was a challenging but rewarding experience, as it required careful synchronization to ensure smooth gameplay. This project helped me strengthen my Java programming skills and taught me how to design systems that handle concurrent tasks efficiently.

Incident Reports Website

This project involved building a web-based incident reporting system with PHP and SQL for the backend. The system allowed users to log, retrieve, and analyze reports securely. I designed a database schema to store incident data and implemented input validation to ensure data integrity. Developing the user interface taught me how to create intuitive and user-friendly designs, while working on the backend reinforced my understanding of database management and secure data handling. This project was a great opportunity to apply full-stack development skills to solve a real-world problem.
