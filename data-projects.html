Data Analysis

The source code for the projects detailed on this page is available in my GitHub repository. Each project represents a real-world data problem worked through from raw data to actionable insight.

S&P 500 Financial Analysis

In this project, I analyzed S&P 500 financial data to surface performance trends, sector comparisons, and key market indicators using Tableau. The dashboard provides an interactive view of market performance, allowing exploration across dimensions including sector breakdowns, historical trends, and relative stock performance. Building this project strengthened my skills in visual storytelling with data and designing dashboards that communicate findings clearly to both technical and non-technical audiences.

Credit Card Transaction Analysis

In this project, I analyzed 555,719 real credit card transactions to surface customer spending behavior, merchant category performance, geographic revenue distribution, and month-over-month financial trends. I wrote advanced SQL queries using window functions, CTEs, correlated subqueries, and ROLLUP aggregation to extract meaningful patterns from large-scale transactional data directly applicable to banking, finance, and business needs. I also produced executive-level financial summaries segmenting customers by lifetime value, reinforcing how to frame technical findings for a business audience.

NYC 311 Service Request Dashboard

For this project, I cleaned and restructured over 4 million NYC 311 service requests, engineering resolution time metrics and time-based features to enable performance analysis across all 5 boroughs. The scale of the dataset made thoughtful data cleaning and feature engineering essential before any meaningful analysis could begin. The final deliverable was a two-page Power BI dashboard uncovering borough-level demand patterns, agency resolution performance, and the complaint types with the longest average resolution times. This project gave me practical experience translating a messy real-world dataset into a polished business intelligence product.

Video Game Data Analysis & Recommendation Model

In this project, I built a machine learning pipeline to recommend similar video game titles and predict critic score ratings based on user input. I preprocessed a video game sales dataset by standardizing and encoding features across platform, genre, sales figures, and critic scores to prepare clean inputs for modeling. Through iterative feature engineering and model tuning, I improved prediction accuracy by 25%, demonstrating how thoughtful data preparation directly impacts model performance.
